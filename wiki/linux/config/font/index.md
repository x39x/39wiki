## serif

### 西文

(latin modern)[https://www.gust.org.pl/projects/e-foundry/latin-modern/download]

下载到的字体有不同的数字后缀，分别为不同情况优化

| 后缀 | 设计特点                            |
| ---- | ----------------------------------- |
| 5–7  | 脚注、公式下标 笔画更粗、结构更清晰 |
| 8–9  | 小正文 字间距更大                   |
| 10   | 标准正文                            |
| 12   | 标题/强调正文                       |
| 17   | 大标题                              |

字体不全，noto serif fallback
